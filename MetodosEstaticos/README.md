Criar uma classe com métodos estáticos para calcular:
- Dado 2 números, retorna o maior deles;
- Dada uma temperatura em graus celsius, retorna o valor em fahrenheit;
- Dado 2 números, retornar um número aleatório entre os 2 (ex: passados 1 e 4, pode retornar aleatoriamente 1, 2, 3 ou 4);
- Dado um número de CPF, retorna TRUE caso seja um número válido (validando inclusive seu digito verificador) ou FALSE caso contrário;
- Dado um número de CPF (não formatado), retorna uma String com CPF formatado (ex: dado 30723401888 retorna 307.234.018-88);
